## Como Publicar no GitHub Pages

1. Faça upload de todos os ficheiros para um repositório GitHub
2. Vá a **Settings → Pages**
3. Em **Source**, selecione `Deploy from a branch`
4. Escolha `main` e a pasta raiz `/`
5. Clique **Save**

O site ficará disponível em `https://[username].github.io/[repo-name]/`
